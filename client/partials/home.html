Bem-Vindo

O que é o sentionário?

O sentionário é uma plataforma que utiliza a colaboração para coletar e classificar expressões de sentimento. Através dos dados coletados, podemos gerar uma base de informação para que computadores estimem como uma pessoa se sente ao ser exposta à uma frase/texto.

Para que serve?

Essa plataforma pode ajudar pesquisadores da área de análise de sentimentos a criar algoritmos capazes de identificar o sentimento presente em um texto. (Por exemplo, identificar se um comentário em uma página é positivo ou negativo)

Como colaborar?

Para contribuir com a plataforma você precisa criar sua conta nesta página e depois informando como você se sente ao ver determinadas palavra.
