O que é o sentionário?

O sentionário é uma plataforma que utiliza a colaboração para coletar e classificar palavras quanto ao sentimento. Através dos dados coletados, é possível gerar uma base para consultar o sentimento em cada palavra.

Para que serve?

Essa plataforma pode ajudar pesquisadores da área de análise de sentimentos a criar algoritmos capazes de identificar o sentimento presente em um texto. (Por exemplo, identificar se um comentário em uma página é positivo ou negativo)

Como colaborar?

Para contribuir com a plataforma você precisa criar sua conta nesta página e depois clicar em "Classificar Palavras" no menu superior.
